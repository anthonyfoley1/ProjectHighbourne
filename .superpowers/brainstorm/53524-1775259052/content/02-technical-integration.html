How should technicals relate to your existing RV view?

You already have per-ticker RV charts with z-scores. How should the technical layer fit in?

A

New Tab — Separate Technical Chart

A dedicated "Technicals" tab with its own price chart showing RSI, MAs, Bollinger bands, volume. RV and technicals stay independent views you flip between.

B

Overlay on RV Chart

Add technical signals (overbought/oversold markers, MA crossovers) directly onto the existing RV ratio chart. One view tells you "cheap AND oversold" at a glance.

C

Stacked Panels — Same Page

RV chart on top, price chart with technicals below, sharing the same time axis. You see both simultaneously without switching tabs, but each chart stays clean.
