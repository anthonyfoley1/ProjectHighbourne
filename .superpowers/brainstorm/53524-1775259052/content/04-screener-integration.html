How should technicals show up in the Screener tab?

Your screener currently ranks stocks by RV z-score. How should technical signals integrate?

A

Add Columns

Add RSI, MACD signal, and MA trend columns to the existing screener table. You can sort/filter by any column. Simple and flexible.

B

Composite Score

Combine RV z-score + technical signals into a single "opportunity score." One number that ranks stocks by how attractive they are across both lenses.

C

Traffic Light Icons

Keep RV z-score as the primary sort, but add colored signal dots next to each ticker: green = oversold + bullish momentum, red = overbought + bearish. Quick visual scan.

D

Both A + C

Add the raw data columns (sortable/filterable) AND the traffic light summary icons. Full data access plus at-a-glance scanning.
